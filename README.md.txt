## The Core Framework (Reflected in the Site)

### The Four Realms
1. **Original Light (Glory)** — Adam's pre-fall state: peace, belonging, communion with God
2. **The Turn** — Fear, shame, hiding, blame (Genesis 3:10-12)
3. **The Shadow** — Spiritual death, contained sorrows, "spiritual vampire" state
4. **Restored Glory** — Return to light, honest self-reflection, growing up

### The Lamed Heart Pivot
- Hebrew letter #12 (ל) = Autosome 12 = the heart bridge
- Letters 1–11: shadow/survival realm
- Letter 12 (Lamed): the pivot/decision point
- Letters 13–22: spirit/restored glory realm
- This is Crystal's signature coaching concept

### The DNA of the Heart
- 22 Hebrew letters = 22 human autosomes
- Original framework by Crystal C. Bingham
- Source texts: 1611 KJV, Apocrypha (Wisdom of Solomon, 2 Esdras), Dead Sea Scrolls (1QS 4:5)
